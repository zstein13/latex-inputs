\section{\StsTestID}
\label{loc:Prep\StsTestID}
% \input{DIDINFO_Snippets/STS/STS_3.x.0_DIDINFO.tex}

Test \StsTestID{WiFi Packet Test} will validate the a Raspberry Pi Pico W has successfully connected to the internet.

\subsection{Hardware Preparation}
\label{loc:HardwarePrep\StsTestID}
% \input{DIDINFO_Snippets/STS/STS_3.x.1_DIDINFO.tex}

This hardware required for test \StsTestID{WiFi Packet Test} is a Raspberry Pi Pico W.

\subsection{Software Preparation}
\label{loc:SoftwarePrep\StsTestID}
% \input{DIDINFO_Snippets/STS/STS_3.x.2_DIDINFO.tex}

This software preparation for test \StsTestID{WiFi Packet Test} are the following:

\begin{itemize}
    \item Raspberry Pi Pico W Firmware to connect the Pico W to the internet.
    \item A Python script to send packets across a WiFi network to the Raspberry Pi Pico W.
\end{itemize}

\subsection{Other Preparation}
\label{loc:OtherPrep\StsTestID}
% \input{DIDINFO_Snippets/STS/STS_3.x.3_DIDINFO.tex}

This other preparation for test \StsTestID{WiFi Packet Test} is to obtain the WiFi SSID and WiFi password for the network and to obtain the IP Address for the Raspberry Pi Pico W.
