\section{\StsTestID}
\label{loc:Prep\StsTestID}
% \input{DIDINFO_Snippets/STS/STS_3.x.0_DIDINFO.tex}

Test \StsTestID{Environmental Threshold Test} will validate that the irrigation system is activated/deactivated when certain environmental thresholds are measured.

\subsection{Hardware Preparation}
\label{loc:HardwarePrep\StsTestID}
% \input{DIDINFO_Snippets/STS/STS_3.x.1_DIDINFO.tex}

This hardware required for test \StsTestID{Environmental Threshold Test} are a water pump, temperature sensor, soil moisture sensor, and a Raspberry Pi Pico W.

\subsection{Software Preparation}
\label{loc:SoftwarePrep\StsTestID}
% \input{DIDINFO_Snippets/STS/STS_3.x.2_DIDINFO.tex}

This software preparation for test \StsTestID{Environmental Threshold Test} is firmware
that monitors the temperature and soil moisture data and activates/deactivates the water pump once environmental thresholds have been met.

% \input{DIDINFO_Snippets/STS/STS_3.x.3_DIDINFO.tex}
