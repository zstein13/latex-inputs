\DIDINFO{SPS/SSS-3.8.0 :: This section shall specify the system requirements, if any, concerned with maintaining security and privacy. 
The requirements shall include, as applicable, the security/privacy environment in which the system must operate, the type and degree of security or privacy to be provided, the security/privacy risks the system must withstand, required safeguards to reduce those risks, the security/privacy policy that must be met, the security/privacy accountability the system must provide, and the criteria that must be met for security/privacy certification/accreditation.}
