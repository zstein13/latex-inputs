\DIDINFO{STP-4.2.x.y :: This paragraph shall identify a test by project-unique identifier and shall provide the information specified below for the test. 
Reference may be made as needed to the general information in Section 4.1.
\begin{itemize}
\item Test objective
\item Test level
\item Test type or class
\item Qualification method(s) as specified in the requirements specification
\item Identifier of the CSCI requirements and, if applicable, software system requirements addressed by this test. (Alternatively, this information may be provided in Section 6.)
\item Special requirements (for example, 48 hours of continuous facility time, weapon simulation, extent of test, use of a special input or database)
\item Type of data to be recorded
\item Type of data recording/reduction/analysis to be employed
\item Assumptions and constraints, such as anticipated limitations on the test due to system or test conditions--timing, interfaces, equipment, personnel, database, etc.
\item Safety, security, and privacy considerations associated with the test.
\end{itemize}
